Nav


Navigation is a very important element of a page, it helps to move between pages, useful when you have a lot of links. Note: Nav requires Jquery. See examples below:


Note: Our page is using Sidnav and Topnav (in other form, class using in our sidenav is not in download). Plese resize screen and see effect.

Note: If u want "fixed" Topnav use topnav-fixedclass in <span class="topnav-show">.
Note: If u use topnav-fixed class u must add <div class="main"> before Nav.

<!-- JQUERY (REQUIRED) -->
<script src="https://igorkowalczyk.github.io/fest/download/jquery.min.js"></script>
<!-- /JQUERY (REQUIRED) -->

<!-- SIDE NAV AND TOPNAV-->
<div class="grid-container">
<div class="menu-icon">
<i class="fas fa-bars header-menu"></i>
</div>
<span class="topnav-show">
Example logo
<button class="ripple ripple-effect">Example Button</button>
</span>
<div class="sidenav">
<div class="sidenav-close-icon">
<i class="fas fa-times sidenav-brand-close"></i>
</div>
<ul class="sidenav-list">
<a href="#!" class="ripple ripple-effect link-nav">Link 1</a>
<a href="#!" class="ripple ripple-effect link-nav">Link 2</a>
</ul></div></div>
<!-- /SIDE NAV AND TOPNAV-->

Note: The sidenav requires Jquery. If u don't like Jquery see below.


Or Without Jquery

JS
/* IN .JS OR IN <SCRIPT></SCRIPT> */

/* WITHOUT JQUERY */

const menuIconEl = document.querySelector('.menu-icon');
const sidenavEl = document.querySelector('.sidenav');
const sidenavCloseEl = document.querySelector('.sidenav-close-icon');
const onclick = e => sidenavEl.classList.toggle( 'active' );
menuIconEl.addEventListener( 'click', onclick );
sidenavCloseEl.addEventListener( 'click', onclick );

/* /WITHOUT JQUERY */

/* /IN .JS OR IN <SCRIPT></SCRIPT> */

Top Nav


<!-- TOP NAV -->
<div class="grid-container">
<span class="topnav-show">
Example logo
<button class="ripple ripple-effect">Example Button</button>
</span>
</div>
<!-- /TOP NAV -->
Note: You should use sidenav with topnav. This helps to keep the site responsive.

Sicky


After adding the sticky class, the nav will "stick" to the screen while scrolling through the page.

<!-- SICKY NAV -->
<div class="grid-container sticky">
<span class="topnav-show">
Example logo
<button class="ripple ripple-effect">Example Button</button>
</span>
</div>
<!-- SICKY NAV -->
Note: Dont use sicky class in "fixed" topnav.
