Range


Are you bored with default zippers? We want to change this, so in our .css file we have prepared a special style, and in the .js file a special counter that makes it easier to use the slider. See examples below:
50
250 

<!-- RANGE -->
<div class="range-slider">
<input class="range-slider-range" type="range" min="0" max="100" value="50">
<span class="range-slider-value">50</span>
</div>
<div class="range-slider">

<input class="range-slider-range" type="range" min="0" max="500" value="250">
<span class="range-slider-value">250</span>
</div>
<!-- /RANGE -->
