Preloader


If you have content that will take a long time to load, you should use a preloader that will show the progress of loading the content. See examples below:


Default Spinner


<!-- DEFAULT SPINNER -->
<div class="spinner spinner-large"></div>
<div class="spinner spinner-medium"></div>
<div class="spinner spinner-small">
<!-- /DEFAULT SPINNER -->

Double Spinner


<!-- DOUBLE SPINNER -->
<div class="spinner-2 spinner-large"></div>
<div class="spinner-2 spinner-medium"></div>
<div class="spinner-2 spinner-small"></div>
<!-- /DOUBLE SPINNER -->
Size the spinners by adding spinner-SIZEclass. spinner-large - Large Spinner, spinner-medium - Medium Spinner, spinner-small - Small Spinner.

Colored Default Spinner


<!-- COLORED DEFAULT SPINNER -->
<div class="spinner spinner-red spinner-large"></div>
<div class="spinner spinner-green spinner-medium"></div>
<div class="spinner spinner-blue spinner-small"></div>
<!-- /COLORED DEFAULT SPINNER -->

Color the default spinner by adding spinner-COLORclass. spinner-red - Red Spinner, spinner-green - Green Spinner, spinner-blue - Blue Spinner.

Colored Double Spinner


<!-- COLORED DOUBLE SPINNER -->
<div class="spinner-2 spinner-2-red  spinner-large"></div>
<div class="spinner-2  spinner-2-green spinner-medium"></div>
<div class="spinner-2 spinner-2-blue spinner-small"></div>
<!-- /COLORED DOUBLE SPINNER -->
Color the double spinner by adding spinner-2-COLORclass. spinner-2-red - Red Spinner, spinner-2-green - Green Spinner, spinner-2-blue - Blue Spinner.


Progress Bar




Indeterminate



<!-- INDETERMINATE -->
<div class="progress">
<div class="indeterminate"></div>
</div>
<!-- /INDETERMINATE -->

Determinate



<!-- DETERMINATE -->
<div class="progress">
<div class="determinate" style="width: 84%"></div>
</div>
<!-- /DETERMINATE -->
