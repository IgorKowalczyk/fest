Typewriter


Typewriter is responsible for the effect of "writing".
<!-- TYPEWRITER -->
<span class="typewriter" data-period="2000" data-type='[ "Aurolia.css", "Demo of Typewriter" ]'>
<span class="typewriter-line"></span>
<!-- /TYPEWRITER -->
data-period="TIME IN MS"- responsible for the time after which the whole action will be repeated.
data-type='[ "TEXT"]'- responsible for the message shown.

Custom text cursor

/* IN .CSS OR IN <STYLE></STYLE> */

/* /CUSTOM TYPEWRITER */

.typewriter > .typewriter-line {
border-right: 0.08em solid #000 !important
}

/* /CUSTOM TYPEWRITER */

/* /IN .CSS OR IN <STYLE></STYLE> */

<span class="badge badge-blue new">12</span>

<span class="badge badge-red" data-badge-caption="custom caption">12</span>
<span class="badge badge-blue" data-badge-caption-before="custom caption">12</span>
<!-- /BADGES -->
new- add a "new" keyword and background.
badge-blueandbadge-red- add custom background.
data-badge-caption="TEXT"- Custom caption after badge.
data-badge-caption-before="TEXT"- Custom caption before badge.
